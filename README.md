# ★ AIRS Studio — Governed AI Nexus Generator
### DeBacco Nexus LLC · USPTO Patent Pending 19/571,156

AIRS Studio is the world's first governed AI on-demand data site builder. 
Users describe their domain problem — AIRS Studio generates and deploys a complete, 
governed data infrastructure (a Nexus) powered by the IGM.

## Architecture

```
User submits domain prompt
  → IGM classifies + validates (pre-inference governance)
  → Claude API generates complete Nexus codebase
  → Railway API provisions project + PostgreSQL
  → Generated code deployed automatically
  → User receives live Nexus URL
  → Node builds custom Pods on demand
```

## The Nexus · Node · Pod Model

- **Nexus** — Your governed data core. Generated by AIRS Studio.
- **Node** — The AI builder engine. Generates custom Pod attachments.
- **Pods** — Windows, rooms, doorways, dropdowns, infographics. Self-designed.

## Setup

1. Clone this repository
2. Copy `.env.example` to `.env` and fill in your keys:
   - `ANTHROPIC_API_KEY` — your Anthropic API key
   - `RAILWAY_API_TOKEN` — your Railway API token
   - `RAILWAY_TEAM_ID` — your Railway team ID
3. Install dependencies: `npm install`
4. Run locally: `npm start`
5. Deploy to Railway: `railway up`

## Environment Variables

| Variable | Description |
|---|---|
| `ANTHROPIC_API_KEY` | Claude API key for Nexus generation |
| `RAILWAY_API_TOKEN` | Railway API token for deployment |
| `RAILWAY_TEAM_ID` | Railway team/account ID |
| `ENCRYPTION_KEY` | AES-256 key for PII encryption |
| `PORT` | Server port (default 3000) |

## API Endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/api/generate` | POST | Generate + deploy a new Nexus |
| `/api/status/:id` | GET | Check Nexus generation status |
| `/api/nexus` | GET | List all Nexus deployments |
| `/api/pod/build` | POST | Build a Pod for a deployed Nexus |
| `/api/nexus/:id/download` | GET | Download Nexus as deployable zip |
